校园二手交易平台系统说明文档
一、功能简介
校园二手交易平台是面向师生的闲置物品交易系统，支持商品发布、浏览查询、在线沟通及交易管理，助力校园资源循环利用。
二、环境配置
（一）开发环境
JDK：1.8 及以上版本
Maven：3.6 及以上版本
MySQL：8.0 及以上版本
Tomcat：8.5 及以上版本
（二）技术栈
后端：Spring MVC 5.3.30 + MyBatis 3.5.13 + MySQL 8.0
前端：Bootstrap 5.1.3 + JSP
工具：Maven 3.8.1（构建）、HikariCP 4.0.3（连接池）

四、部署步骤
（一）打包与部署
编译打包：
在项目根目录执行命令：
bash
mvn clean package  

生成 target/campus-market.war 包。
部署到 Tomcat：
将 war 包复制到 Tomcat 的 webapps 目录，启动 Tomcat 后自动解压。
数据库配置：
修改 src/main/resources/database.properties：
properties
jdbc.url=jdbc:mysql://localhost:3306/campus_trading?useUnicode=true&characterEncoding=UTF-8  
jdbc.username=root  
jdbc.password=123456  

执行 database/init.sql 初始化表结构及测试数据。
（二）性能优化
连接池配置：在 applicationContext.xml 中调整 HikariCP 参数（如最大连接数 maximumPoolSize=10）。
MyBatis 缓存：在 Mapper 文件中添加 <cache/> 标签启用二级缓存。
静态资源优化：将 CSS/JS 上传至 CDN，修改前端引用路径（如 src="https://cdn.example.com/style.css"）。
Tomcat Gzip 压缩：在 server.xml 中配置：
xml
<Connector port="8080" compression="on" compressionMinSize="2048"  
           compressableMimeType="text/html,text/css,application/javascript"/>  


五、启动验证
部署完成后，访问 http://localhost:8080/campus-market，系统自动跳转到登录页，使用测试账号（如 username: user1 / password: user123）登录后即可体验商品发布、浏览等功能。
